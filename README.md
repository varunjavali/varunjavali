## 🚀 Projects

### 🛒 [Shopwave — Microservices E-Commerce Platform](https://github.com/varunjavali)
> AWS EKS · Terraform · Kubernetes · Docker · Python · Node.js · Go · React · MongoDB

Full-stack e-commerce system on **microservices architecture** — Auth, Products, Orders, Payments & Notifications as separate services. Infrastructure provisioned on **AWS via Terraform** (VPC + EKS); services containerized with **Docker**, deployed to **Kubernetes** behind an Nginx API Gateway. React + Redux frontend with full checkout flow.

---

### 📚 [LSM — Learning Management System](https://github.com/varunjavali)
> Node.js · Express · Prisma ORM · PostgreSQL · React · Vite · Tailwind CSS · JWT

Full-stack LMS with **role-based access** (Admin / Trainer / Student). JWT session auth, course & video management, per-video watch progress tracking. Prisma ORM on PostgreSQL backend; React/Vite/Tailwind frontend with protected routes and dashboards.

---

### 🛍️ VCart — Full-Stack eCommerce System
> Flutter · Firebase · Firestore · Provider · Google Auth · OpenStreetMap

Two-app system sharing a single Firebase backend:

| App | Description |
|-----|-------------|
| **User App** | Google Auth, product browse/search/filter, wishlist, cart (Provider), coupon engine, wallet + cashback, order tracking (pending → shipped → delivered), OpenStreetMap address picker |
| **Admin Panel** | Flutter Windows desktop app — product/order/user management, real-time Firestore streams, live order status updates |

---

### 🚪 [EduSense Gate — Smart Attendance System](https://github.com/varunjavali)
> Python · Streamlit · Flask · SQLite · Wi-Fi Network Scanning

Wi-Fi-based attendance system that **auto-detects students** on an authorized network using live device scanning. Faculty dashboard for session management and attendance approval; CSV export for records.

### 🤖 [DeepFake Detection System](https://github.com/varunjavali)

AI-powered deepfake classifier using an **ensemble of deep learning models** with FFT frequency analysis and metadata inspection to distinguish real media from AI-generated content.
